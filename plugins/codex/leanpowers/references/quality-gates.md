# Quality gates

These invariants are always active:

1. Do not claim completion without current evidence.
2. Diagnose an unknown failure to its root cause before claiming a repair.
3. Behavior changes require appropriate regression evidence.
4. Stay inside the declared scope; re-shape material expansion.
5. High-risk work requires an independent review.
6. Destructive, irreversible, credential-gated, or production actions require authorization.
7. Re-evaluate when new evidence contradicts the current conclusion.
8. Report every material validation gap explicitly.

Workflow preferences change ceremony, never these gates.
